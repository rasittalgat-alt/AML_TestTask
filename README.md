# AML_TestTask — классификация наименований ТРУ

Решение тестового задания на позицию ML-инженера.

Проект содержит:
- подготовку сэмпла из большого CSV;
- выборку для ручной разметки категорий;
- обучение модели (TF-IDF + линейный классификатор);
- скрипт для инференса и применения модели к новым данным.

Подробности и инструкции по запуску я описал внутри кода и в комментариях.
